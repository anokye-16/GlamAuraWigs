# Wireframes for Front-End Pages

## Homepage
- Logo, navigation bar, featured products, call-to-action buttons.

## Product Catalog
- Grid/list of products with images, names, prices, and 'Add to Cart' buttons.

## Shopping Cart
- List of selected products, quantities, total price, checkout button.

## Checkout Page
- Form for shipping and payment details, order summary, place order button.

## Customer Dashboard
- User info, order history, profile management links.

## Admin Dashboard
- Links to manage products, orders, users, reports.
