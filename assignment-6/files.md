# Files for Assignment 6: Back-End Development (Business Operations System)

## Back-End Module Files

| File | Module | Purpose |
|------|--------|---------|
| backend_system.html | ALL MODULES | Complete interactive back-end dashboard |
| crm_feedback.php | CRM | PHP handler for customer feedback submission |
| finance_dashboard.php | Finance | PHP finance data display |
| supply_chain.php | Supply Chain | PHP supply chain management |
| db.php | All | MySQL connection configuration |
| login.php | Auth | User login processing |
| signup.php | Auth | User registration |
| logout.php | Auth | Session destruction |
| fetch_products.php | Ordering | Returns products as JSON from DB |
| place_order.php | Ordering | Inserts order + items into DB |
| checkout.php | Ordering | Processes full checkout form |
| get_order.php | Ordering | Returns order details by ID |
| sql.txt | All | MySQL schema: users, products, orders, order_items |
